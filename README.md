# Food Delivery Time Prediction

Welcome to the Food Delivery Time Prediction project repository. This project aims to predict the delivery time for food orders using machine learning techniques. Below is an overview of the project, including features, code snippets, and instructions for running the code.

## Overview

This project involves predicting the delivery time for food orders based on various factors such as delivery person details, weather conditions, traffic density, and more. The dataset used for training and testing contains information about past food deliveries.

---

## Features

- **Data Cleaning**: Handling missing values, data type conversion, and ensuring data consistency.
- **Data Transformation**: Extracting relevant features, encoding categorical variables, and performing feature engineering.
- **Model Building**: Implementing regression models such as Linear Regression, Decision Tree Regression, Random Forest Regression, and XGBoost Regression.
- **Model Evaluation**: Evaluating model performance using metrics like Mean Absolute Error (MAE), Mean Squared Error (MSE), Root Mean Squared Error (RMSE), and R-squared (R2) score.
